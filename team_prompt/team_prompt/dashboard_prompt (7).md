# HTML 기반 정적 대시보드 제작 프롬프트

이 문서는 `nps-company` 프로젝트의 분석 결과를 바탕으로 한 HTML 기반 정적 대시보드를 제작하기 위한 상세 지침을 담고 있습니다.

## 1. 프로젝트 개요
*   **목적**: 국민연금 가입 사업장 데이터를 시각화하고, 주요 기업의 연봉 추정치 및 고지금액 순위를 한눈에 파악할 수 있는 인터랙티브 대시보드 구축.
*   **대상 데이터**: `nps-company/data/국민연금공단_국민연금 가입 사업장 내역_20260423.csv` 기반의 분석 결과.
*   **출력 형식**: 단일 HTML 파일 (CSS 및 JS 포함 또는 CDN 활용).

## 2. 디자인 가이드라인
*   **테마**: Clean & Modern (Light 모드 기반, Blue/Gray 톤 사용).
*   **레이아웃**: Responsive Grid Layout (메인 KPI -> 시각화 차트 -> 데이터 테이블 순).
*   **라이브러리 추천**:
    *   **CSS**: Tailwind CSS (CDN) 또는 Bootstrap 5.
    *   **차트**: Chart.js 또는 ApexCharts.js.
    *   **테이블**: DataTables.net (검색, 정렬, 페이지네이션 기능용).

## 3. 대시보드 구성 요소

### 3.1 상단 KPI 카드 (핵심 지표)
*   총 사업장 수: 582,825개
*   평균 가입자 수: 19.56명
*   평균 당월고지금액: 약 715만 원
*   최대 가입자 수 기업: 삼성전자(주) (125,718명)

### 3.2 시각화 섹션 (이미지 활용 및 차트 구현)
`nps-company/images/` 폴더의 기존 시각화 이미지를 표시하거나, 데이터를 바탕으로 동적 차트를 구현합니다.
*   **지역별 분포**: 시도별 사업장 수 (viz3.png) 및 평균 가입자 수 (viz6.png).
*   **업종 분석**: 상위 30개 업종 빈도 (cat_top30.png).
*   **고용 동향**: 신규 취득자 vs 상실 가입자 상관관계 (viz8.png).
*   **사업장 형태**: 법인 vs 개인 비율 (viz4.png).

### 3.3 기업 데이터 테이블 (인터랙티브)
`eda_report.md`의 데이터를 활용하여 검색 및 정렬이 가능한 테이블을 구성합니다.
*   **TAB 1: 추정 평균 연봉 상위 50 (가입자 1,000명 이상)**
    *   컬럼: 순위, 기업명, 가입자수, 추정평균연봉(만원)
*   **TAB 2: 당월고지금액 상위 50**
    *   컬럼: 순위, 기업명, 가입자수, 당월고지금액(억원)

## 4. 기능 요구사항
1.  **반응형 디자인**: 모바일 및 데스크탑 환경에서 최적화된 레이아웃 제공.
2.  **이미지 모달**: 시각화 이미지 클릭 시 확대해서 볼 수 있는 기능.
3.  **데이터 검색**: 기업명으로 연봉/고지금액 순위를 빠르게 찾을 수 있는 검색 바.
4.  **필터링**: 시도별 또는 업종별로 요약 데이터를 필터링할 수 있는 UI (선택 사항).

## 5. 제작 방법 (단계별)
1.  기본 HTML 구조 설계 (Header, Main, Footer).
2.  Tailwind CSS를 활용한 레이아웃 스타일링.
3.  `eda_report.md`의 표 데이터를 JSON 형태로 변환하여 JS 변수에 할당.
4.  DataTables 라이브러리를 연결하여 기업 순위 테이블 구현.
5.  이미지 태그를 사용하여 `../images/` 경로의 시각화 자료 배치.
6.  최종 결과물을 `nps-company/report/dashboard.html`로 저장.
