# 국민연금 가입 사업장 분석 정적 HTML 대시보드 제작 계획 프롬프트

이 문서는 국민연금공단의 가입 사업장 내역 데이터를 기반으로 한 **단일 파일 기반 정적 HTML 대시보드** 제작을 위한 상세 기획 및 구현 가이드라인을 담고 있습니다.

## 1. 프로젝트 개요
- **목표**: 별도의 서버나 복잡한 빌드 도구 없이 브라우저에서 즉시 실행 가능한 HTML 대시보드 구축.
- **주요 타겟**: 데이터 분석 결과물을 빠르게 공유하고자 하는 연구자 및 실무자.
- **데이터 소스**: `nps-company/report/` 내 생성된 CSV 및 JSON 통계 파일.

## 2. 주요 지표 (KPI Cards)
상단 영역에 배치할 고정 지표 (HTML 요소로 직접 구성):
1. **총 가입 사업장 수**: 582,825개
2. **총 가입자 수**: 약 1,140만 명 (추산치)
3. **당월 총 고지금액**: 약 4.1조 원 규모
4. **평균 고용 규모**: 19.56명 (중앙값 6명)
5. **고용 유동성**: 신규/상실 가입자 증감률 요약.

## 3. 대시보드 레이아웃 (Single Page Tabs)
CSS와 간단한 JavaScript를 사용하여 탭 전환 기능을 구현합니다.

### Tab 1: 시장 총괄 (Overview)
- **핵심 지표 카드**: 상단 그리드 배치.
- **사업장 구조 시각화**: 법인/개인 비중(Pie Chart), 등록/탈퇴 현황(Bar Chart).
- **규모별 분포**: 가입자 구간별 사업장 수 분포도.

### Tab 2: 지역 및 업종 (Region & Industry)
- **시도별 통계**: `table_region_stats.csv` 기반의 가로 바 차트.
- **업종 TOP 20**: 고용 인원 및 고지금액 상위 업종 순위표.
- **텍스트 인사이트**: TF-IDF 분석 결과(주요 키워드)를 태그 클라우드 형태로 배치.

### Tab 3: 고용 동향 (Dynamics)
- **신규 vs 상실 분석**: 월간 고용 변동성 시각화.
- **변수 간 관계**: 가입자수와 고지금액의 상관관계 산점도 이미지(기존 저장된 `images/` 활용).
- **상관관계 히트맵**: `plot10_corr_heatmap.png` 이미지 임베딩.

### Tab 4: 데이터 요약 (Data Table)
- **주요 통계 테이블**: `cat_stats.csv`, `num_stats.csv`의 내용을 가독성 있게 정리한 테이블.

## 4. 기술 사양 (Static Tech Stack)
- **Core**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+).
- **Styling**: `styles.css` 파일을 분리하거나 `<style>` 태그 내에 작성. 별도의 프레임워크 없이 순수 CSS 사용.
- **Visualization**: **Chart.js** 또는 **ApexCharts** (CDN 링크 사용).
- **Icons**: **Lucide Icons** 또는 **FontAwesome** (CDN 사용).
- **Data Loading**: `fetch()` API를 사용하여 `report/` 폴더의 JSON/CSV 파일을 비동기 로딩하거나, 소량의 데이터는 JS 변수에 직접 포함.

## 5. 시각화 및 디자인 가이드
- **테마 컬러**: 국민연금 브랜드와 조화를 이루는 블루(` #1a73e8`) 및 그레이톤 사용.
- **반응형**: `@media` 쿼리를 사용하여 모바일에서도 지표 카드가 세로로 나열되도록 설계.
- **이미지 활용**: 복잡한 계산이 필요한 산점도나 히트맵은 실시간 렌더링 대신 `images/` 폴더의 정적 이미지 파일을 `<img src="...">`로 임베딩.

## 6. 구현 시 주의사항
- **파일 경로**: 모든 리소스(이미지, 데이터)는 상대 경로를 사용하여 로컬 환경에서도 잘 작동하도록 함.
- **성능**: 58만 건 전체 데이터를 로딩하지 말고, 분석 완료된 `report/` 내 요약 파일(CSV/JSON)만 로딩하여 브라우저 부하 최소화.
- **인터랙티브**: 탭 클릭 시 부드러운 전환 효과(Transition) 적용.
- **폰트**: Pretendard 또는 시스템 기본 폰트(San-serif) 사용.
