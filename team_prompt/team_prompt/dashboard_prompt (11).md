# [Prompt] 비즈니스 인사이트 통합 시각화 대시보드 생성

다음 지침에 따라 `nps-company` 프로젝트의 분석 결과를 통합하여 보여주는 **단일 HTML 인터랙티브 대시보드** 코드를 작성해 주세요.

## 1. 기술 제약 사항
- **형태**: 모든 CSS와 JavaScript가 포함된 단일 `.html` 파일
- **프레임워크**: Tailwind CSS (CDN), Chart.js (CDN), Lucide Icons (CDN) 활용
- **데이터**: 아래 제공된 데이터를 JavaScript 상수로 포함하여 독립적으로 작동하게 함
- **디자인**: 다크 모드(Slate/Zinc 계열), 미래지향적 유리 효과(Glassmorphism) 적용

## 2. 대시보드 구성 요소

### 상단: 핵심 지표 (KPI Cards)
1. **NPS 분석 대상**: 582,825개 사업장
2. **소비 성장 지수**: 1.45 (2020년 대비)
3. **HR 분석 규모**: 1,470명 (IBM 데이터 기반)
4. **전체 이직률**: 16.1%

### 섹션 1: 대한민국 경제 지도 (국민연금 데이터)
- **차트**: "보험료 고지금액 Top 10 기업" (Bar Chart, 수평)
- **데이터**: 
    - 삼성전자: 745.2억 / 현대자동차: 381.3억 / SK하이닉스: 205.4억 / LG전자: 201.8억 / LG디스플레이: 134.5억 / 삼성디스플레이: 128.5억 / 쿠팡풀필먼트: 121.1억 / 한국전력공사: 121.1억 / 한국철도공사: 109.4억 / 대한항공: 103.0억
- **인사이트**: 삼성전자가 2위인 현대차의 약 2배 수준으로 압도적 기여.

### 섹션 2: 소비 트렌드 대전환 (카드 이용 트렌드)
- **차트**: "2024년 업종별 성장률 랭킹 (Base 2020=1.0)" (Bar Chart)
- **데이터 (상위)**: 여행(4.71), 항공(4.56), 자동차판매(3.33), 관람(3.31), 사우나/마사지(2.34)
- **데이터 (하위)**: 통신/우편판매(0.55), 홈쇼핑(0.64), 가전(0.71), 교육기관(0.72)
- **인사이트**: 코로나 이후 '비대면'에서 '경험(여행/문화)'으로 소비의 중심이 완전히 이동.

### 섹션 3: 인적 자원 리스크 관리 (HR 이직 분석)
- **차트 1**: "이직 여부에 따른 월 소득 분포" (Box Plot 또는 단순 비교 차트)
- **차트 2**: "시간외근무 여부와 이직률의 상관관계" (Stacked Bar Chart)
- **데이터**: 
    - 시간외근무(Yes): 이직률이 'No' 그룹 대비 현저히 높음
    - 근속연수: 0~5년 차 주니어 그룹에서 이직이 가장 활발함
- **인사이트**: 낮은 보상과 높은 업무 강도가 이직의 핵심 원인.

## 3. UI/UX 상세 지침
- **네비게이션**: 상단 또는 측면에 탭(Tab) 메뉴를 두어 [Overview], [NPS], [Card], [HR] 섹션 전환 가능하게 함.
- **애니메이션**: 페이지 로드 시 차트 애니메이션 적용.
- **이미지**: `/images/` 경로의 기존 시각화 이미지를 캡션과 함께 배치할 수 있는 플레이스홀더 구성.
- **반응형**: 모바일(1컬럼)과 데스크톱(2~3컬럼) 그리드 레이아웃 최적화.
